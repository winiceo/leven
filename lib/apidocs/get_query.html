# API GET Request Query Syntax ### Reserved Query Params **`qt`** query type (find|one|count|findcount|distinct|tree) **`f`** fields **`s`** sort **`sk`** skip **`l`** limit **`p`** populate **`?qt=(qtype)&f=(fields)&s=(sort)&sk=(skip)&l=(limit)&p=(populate)`** ### Search Conditions: **`key={gt}a`** key is greater than a **`key={gte}a`** key is greater or equal than a **`key={lt}a`** key is lower than a **`key={lte}a`** key is lower or equal than a **`key={in}a,b`** at least one of these is in array **`key={nin}a,b`** any of these values is not in array **`key={ne}a`** key is not equal to a **`key={all}a,b`** all of these contains in array **`key={empty}`** field is empty or not exists **`key={!empty}`** field exists and is not empty **`key={null}`** key is equal to null **`key={near}longitude,latitude,maxDistance`** docs near key **`key={:like:}a`** %like% query **`key={like:}a`** like% query **`key={:like}a`** %like query **`key={null}`** field exists and is null **`{or|b}key1={in}a,b&{or|b}key2=c`** or query (b: 'or' group alias) **`key1=N|{empty}`** or query for same key
